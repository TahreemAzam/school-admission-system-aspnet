# School Admission Management System

A web-based School Admission Management System built using **ASP.NET Core MVC**.

## 🚀 Features
- Online student admission form
- Admin dashboard for managing applications
- Notice board management
- Real-time notifications using SignalR
- Role-based authentication (Admin / Student)
- Clean MVC architecture with Repository Pattern

## 🛠️ Technologies Used
- ASP.NET Core MVC (.NET 8)
- C#
- Entity Framework Core
- SQL Server
- SignalR
- Bootstrap
- JavaScript & jQuery

## 📂 Project Structure
- Controllers
- Models
- Views
- Data (Repositories, Migrations)
- wwwroot (CSS, JS, Images)

## 🎯 Purpose
This project was developed as an academic and portfolio project to demonstrate full-stack web development skills using ASP.NET Core.
